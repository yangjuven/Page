Page
====
Page is a simple and responsive theme for [Octopress](http://octopress.org).


Install
-------
    $ cd octopress
    $ git clone git://github.com/yangjuven/Page.git .themes/Page
    $ rake install['Page']
    $ rake generate


Update
------
    $ cd octopress/.themes/Page
    $ git pull
    $ rake install['Page']
    $ rake generate
