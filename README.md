Page
====
Page is a minimal and responsive theme for [Octopress](http://octopress.org).


Install
-------
    $ cd octopress
    $ git clone git://github.com/yangjuven/page.git .themes/Page
    $ rake install['Page']
    $ rake generate


Update
------
    $ cd octopress/.themes/Page
    $ git pull
    $ rake install['Page']
    $ rake generate
